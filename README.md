# WNCC NASA NeSpaceGrant.org

- These projects are supported by the [NASA Nebraska Space Grant](https://www.nespacegrant.org)
- This is the pespository for Western Nebraska Community College NASA Nebraska Space Grant Student Fellowships
- This is part of an ongoing project by Bill Loring (Mentor)
  - Enhancing Computer Science Curriculum using off-the-shelf Robotics Kits to Increase Whole Brain Learning

## NASA Fellowships 24-25 Mars Rover Simulations

This project simulates navigating, gathering and transmitting information from Mars to Earth using Raspberry Pi based [GoPiGo3](https://gopigo.io)'s.
